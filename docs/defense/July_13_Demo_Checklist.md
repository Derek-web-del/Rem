Complete these steps on the live Glendale site **before** the follow-up defense.

## Subjects (guide + schedule + syllabus)

## Faculty photo

- [ ] Admin → Faculties → Edit demo teacher → upload photo → Save

## Portal access badges

- [ ] Admin dashboard shows **System Administrator** badge
- [ ] Faculty pages show **Faculty Portal** badge with advisory/code
- [ ] Student pages show **Student Portal** badge with grade/section

## 10-minute demo script

1. Admin → **Sections** (Grade 7–10 section)
2. Admin → **Curriculum** (DepEd MATATAG PDFs + workflow callout)
3. Admin → **Subjects** (guide, schedule, faculty, syllabus + template)
4. Admin → **Faculties** (advisory section + photo)
5. Admin → **Students** (enroll in section)
6. Faculty → subject (modules / quiz / material)
7. Student → subject access
8. Admin → **Audit Logs** (accountability)

## Talking points if asked

- **Curriculum vs syllabus:** DepEd PDF = national reference; Glendale syllabus = school term plan derived from that guide.
- **Template:** Syllabus template is downloadable; curriculum uses official DepEd documents, not AI-generated forms.
- **Faculty PII:** Faculty advisory roster excludes contact/address; audit logs record profile views.
- **Offline:** Partial scope — quiz resume + PDF cache only (documented in Chapter 6).
